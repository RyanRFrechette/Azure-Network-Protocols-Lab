# Resume Bullets - Azure Network Protocols Lab

- Built and documented an Azure networking lab using Windows and Linux virtual machines, Azure Virtual Network, Network Security Groups, RDP, SSH, DNS, ICMP, TCP testing, and Wireshark.
- Verified private network connectivity between Azure VMs using ping, SSH, and PowerShell TCP port testing.
- Created and tested an Azure Network Security Group deny rule to demonstrate how firewall configuration can block otherwise healthy VM communication.
- Used Wireshark to capture and compare blocked ICMP traffic, restored ICMP request/reply traffic, and DNS query/response traffic.
- Produced screenshot-backed technical documentation explaining each troubleshooting step in clear help desk / junior cloud support language.
- Confirmed Azure cleanup by deleting the lab resource group and saving command-line proof that billing resources were removed.
